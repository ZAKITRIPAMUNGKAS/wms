# 📦 Nama Perusahaan - WMS Premium Dashboard

A high-end, mobile-first Warehouse Management System (WMS) designed for modern startups and scaling businesses. This prototype provides a seamless user experience across desktop and mobile devices, focusing on logistics efficiency and "Nama Perusahaan" branding.

---

## ✨ Features

- **📱 Mobile-First Design**: Optimized for handheld use in warehouses with a floating bottom navigation bar.
- **🖥️ Desktop Excellence**: Sophisticated sidebar navigation for high-density data management.
- **📊 Advanced Analytics**: Visualized reporting for incoming and outgoing goods with integrated charts.
- **🛠️ Master Data Management**: Fully functional client-side CRUD for Products, Customers, Suppliers, and Warehouses.
- **📄 Logistics Flow**: Complete templates for Barang Masuk, Barang Keluar, Invoicing, and Payment tracking.
- **🔒 Premium Authentication**: Stunning, responsive login interface with glassmorphism effects.

---

## 🛠️ Tech Stack

- **Core**: HTML5, Vanilla JavaScript.
- **Styling**: [Tailwind CSS v3](https://tailwindcss.com/) (delivered via CDN).
- **Iconography**: [Phosphor Icons](https://phosphoricons.com/) (Light/Outline variants).
- **Typography**: [Plus Jakarta Sans](https://fonts.google.com/specimen/Plus+Jakarta+Sans) for a modern, professional look.

---

## 🚀 Quick Start

Since this is a static project, you can run it locally without any installation:

1. Clone the repository:
   ```bash
   git clone https://github.com/ZAKITRIPAMUNGKAS/wms.git
   ```
2. Open `index.html` in your preferred web browser.
3. To view the mobile experience, use your browser's **Toggle Device Toolbar** (F12 > Ctrl+Shift+M) and select a mobile device (e.g., iPhone 12 Pro).
